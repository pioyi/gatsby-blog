How To Get Started With Python

Published during October 13, 2020


Here are some quick tips to get you started with python! You will write some code using VS Code!

Python is my favorite programming language. It's a great and powerful language for every skill level!

Downloading VS Code

I would definitely recommend VS Code as your text editor! It's clean, minimalistic and has a wide variety of features and extensions. You can start here!

Installing Python

You then have to install python on your computer. Open your browser and go to the python installer page. Select your Operating System and the latest python version!

Running The Installer

Download the file and run the installer! A little window should then pop up. Do not forget to select the "Add Python 3.x to PATH" property! It's necessary for python to install correctly!

That should be it! You can now start writing your very own python code! YouTube provides a wide variety of great tutorials to get you started! I personally recommend this amazing youtube channel.
