How It Was Made

Published during October 14, 2020


A brief showcase of all the web technologies which I implemented in order to create this website.

Welcome to my blog! This article is all about the technical side of this site. Although it's a small project I would like to share its structure.

React.js

React.js is considered a "A JavaScript library for building user interfaces". I love its component based logic, which makes JavaScript extremely flexible and organized into distinct folders.

Gatsby.js & Contentful

All the data (Blog Posts) is managed by these two technologies. Contentful is "the modern way to manage content". I can create all of my pages with ease and with the help of GraphQl I can create simple queries to fetch all that data directly into a react template.

That's About It!

Creating full-stack websites nowadays can be extremely simple! All you need is proper documentation and some spare time to invest into programming.

As for the hosting, I tend to use Github Pages for all of my static projects.
